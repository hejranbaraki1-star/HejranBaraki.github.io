📝 Exercise 9: File Handling

Description: Reading from and writing to files in C.

Writing to a File:

#include <stdio.h>

int main() {
    FILE *file = fopen("data.txt", "w");
    if(file != NULL) {
        fprintf(file, "Hello, this is a file!\n");
        fclose(file);
        printf("File created successfully!\n");
    }
    return 0;
}

Reading from a File:

FILE *file = fopen("data.txt", "r");
if(file != NULL) {
    char line[100];
    while(fgets(line, sizeof(line), file)) {
        printf("%s", line);
    }
    fclose(file);
}
