Exercise 4: Loops in C

Description: Using for, while, and do-while loops.

For Loop Example:

#include <stdio.h>

int main() {
    // For loop - print numbers 0 to 4
    for(int i = 0; i < 5; i++) {
        printf("%d ", i);
    }
    printf("\n");
    return 0;
}

While Loop Example:

int i = 0;
while(i < 5) {
    printf("%d ", i);
    i++;
}
