Education & Learning Experience

My education and learning experience in programming has been a gradual journey from understanding basic ideas to building complete pages and solving practical problems. At the beginning, programming felt like a language with many small rules that had to be memorized.

After more practice, I learned that the most important part is not only remembering syntax, but also understanding how to think step by step. In web programming, I started with the structure of an HTML document, including headings, paragraphs, lists, links, images, tables, forms, and semantic sections.

After HTML, I learned how CSS changes the appearance and layout of a page. CSS gave me the ability to control colors, spacing, typography, borders, backgrounds, grids, and responsive design. This final exam website uses those lessons by separating structure from style.

Python also became an important part of my learning journey because it introduced me to a versatile and powerful programming language used in many fields such as web development, data science, automation, and artificial intelligence. Python's simple and readable syntax made it easier for me to understand core programming concepts like variables, data types, loops, functions, and object-oriented programming.

My experience in C programming supported my web programming learning as well. C taught me discipline because it requires careful attention to variables, data types, loops, arrays, functions, pointers, and memory. The combination of both courses helped me become more confident.

Another important part of my learning experience is understanding how websites are published. GitHub Pages is a useful tool because it allows a student to make a project available online. Publishing a site teaches more than coding. It teaches responsibility, organization, version control, and testing.
