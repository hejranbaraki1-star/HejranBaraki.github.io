Exercise 5: Arrays and Strings

Description: Working with arrays and strings in C.

Array Example:

#include <stdio.h>

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    
    for(int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    return 0;
}

String Example:

char name[] = "John";
printf("Hello %s", name);
